# 首诊报告人工质检清单

## 基础信息

- 客户编号是否正确。
- 客户名称是否正确。
- 平台是否正确。
- 类目是否正确。
- 数据周期是否正确。

## 数据口径

- 是否有缺失字段。
- 是否有明显异常值。
- 支付金额、订单数、客单价是否逻辑一致。
- 访客数、订单数、转化率是否逻辑一致。
- 投放消耗、投产比是否口径一致。

## 诊断结论

- 是否把可能原因写成确定原因。
- 是否存在过度承诺。
- 是否有凭空判断。
- 是否对数据缺失做了边界说明。

## 行动建议

- 是否具体。
- 是否能在一周内执行。
- 是否有验证指标。
- 是否符合客户平台和类目。

## 防反推

- 不出现 Coze、Notion、GitHub Actions、豆包、火山方舟等内部工具链说明。
- 不出现 Prompt 原文。
- 不出现规则阈值公式。
- 不出现内部模块名。

## 交付前最终检查

确认报告能回答：

1. 这家店当前最大问题是什么？
2. 下周先做什么？
3. 为什么建议进入4周经营周报试点？
